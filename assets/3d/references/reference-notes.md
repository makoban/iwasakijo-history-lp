# 岩崎城 現在外観3D 参考画像メモ

## 参照方針

- LPの3Dモデルは、現在の岩崎城址公園に建つ城型展望塔を理解するための外観模型。
- 実測復元や文化財調査図面からの厳密な再現ではない。
- 写真は大判転載せず、形状観察と出典確認に使う。
- Google Earthなどの3D表示は、建物の向きや周辺地形を確認する補助に留める。3D形状の抽出、トレース、変換は行わない。

## モデルへ反映した要素

- 前面の石垣と階段。
- 白壁の横長い主棟。
- 屋根の重なり、軒下の垂木、瓦列の粒感。
- 上層展望部の張り出し、手すり、縦格子。
- 正面入口の暗い開口、手すり、標柱。
- 側面の舗装広場、低い木柵、外灯、側面側の格子。
- 屋根の白い縁取り、丸瓦、三角の破風風ディテール。

## 注意

- 写真で見えにくい側面・背面、細部寸法、内部構造は推定。
- さらに精度を上げる場合は、現地で正面・左右側面・背面・高台からの斜め写真を撮影し、同じ焦点距離に近い画像を複数枚そろえる。
