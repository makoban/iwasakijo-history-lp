# 岩崎城 歴史ビジュアル作成メモ

作成日: 2026-05-16

## 共通方針

- 現在の展望塔を、戦国期の天守として扱わない。
- 戦国期の岩崎城は、土塁・空堀・土橋・木柵を中心にした「土の城」として表現する。
- 城下は大都市や華美な城下町にせず、街道沿いの小さな市場として扱う。
- 戦闘場面は流血・炎上・過度な英雄化を避け、地形と足止めの意味を重視する。
- 細部を断定できない建物、人物配置、城下の具体景観は、理解用の再現イラストとして扱う。
